# 图片类
## image

```python
add_image(self,pos:tuple,width=None,height=None,state='fill',imgfile=None,anchor='nw')
```

- pos::位置
- width::宽度（默认为图片宽度）
- height::高度（默认为图片高度）
- state::尺寸状态（none左上角裁剪，fill填充，uniform等比缩放）
- imgfile::图片文件，支持静态gif和png
- anchor::对齐方向

绘制一个静态图片。

返回值：img

> img::图片画布对象，相当于大多数的 `uid`
