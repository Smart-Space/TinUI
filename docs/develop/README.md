# TinUI开发

## 依赖

目前，TinUI没有依赖项目。

## 开源协议

TinUI使用GPLv3作为开源协议。

对于商业软件，需要添加TinUI的gpl-3.0.md的开源许可，为自身使用TinUI做出声明，并且公开TinUI部分的代码。此外，需要注明开发者Smart-Space（Junming Zhang）对TinUI的版权所有，以及GitHub/TinUI的开源代码库：https://github.com/Smart-Space/TinUI/

> 自TinUI-**5.11**起，TinUI支持LGPL-3中的额外许可
